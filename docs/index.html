Installation

Gentoo (Recommended)

# Add the Patronus overlay eselect repository add patronus git https://github.com/yourusername/patronus-gentoo emaint sync -r patronus # Install Patronus emerge net-firewall/patronus # Enable and start services rc-update add patronus default rc-service patronus start # Access web UI at http://localhost:8080 # Default credentials: admin/admin (change immediately!)

From Source

# Clone the repository git clone https://github.com/yourusername/patronus.git cd patronus # Build all crates cargo build --release --workspace # Run the web interface cargo run -p patronus-web # Access at http://localhost:8080

Requirements

  • Linux kernel 5.10+ (eBPF/XDP support)
  • Rust 1.70+ (for compilation)
  • libmnl, libnftnl (for netfilter)
  • Root privileges (for network operations)
